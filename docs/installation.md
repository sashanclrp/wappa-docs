# Installation Guide

*Placeholder for detailed installation instructions*

## System Requirements

- Python 3.12+
- WhatsApp Business API credentials
- Optional: Redis for caching

## Installation Methods

### Using uv (Recommended)

```bash
uv add wappa
```

### Using pip

```bash
pip install wappa
```

### Using Poetry

```bash
poetry add wappa
```
